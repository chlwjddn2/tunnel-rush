# 터널 러시

회전하는 우주 터널을 달리는 3D 엔드리스 러너 게임입니다.  
구멍에 빠지지 않고 최대한 멀리 달리며 젬을 모아보세요.


## 조작법

| 입력 | 동작 |
|------|------|
| `←` / `A` | 왼쪽 레인 이동 |
| `→` / `D` | 오른쪽 레인 이동 |
| `Space` / `↑` / `W` | 점프 |
| `Enter` | 게임 시작 / 재시작 |

모바일·터치 기기에서는 화면 하단에 터치 버튼이 자동으로 표시됩니다.

## 게임 규칙

- 터널은 원형 단면으로 구성되며 12개의 레인이 있습니다.
- 세그먼트마다 일부 레인에 구멍이 생성됩니다. 구멍에 닿으면 게임 오버입니다.
- 거리가 멀어질수록 속도가 빨라지고 구멍 확률도 높아집니다.
- 안전 레인에 가끔 젬(💎)이 등장합니다. 같은 레인에 위치하면 자동으로 수집됩니다.
- 최고 기록은 브라우저 `localStorage`에 저장됩니다.

## 프로젝트 구조

```
game/
├── index.html        # 마크업 및 importmap
├── css/
│   └── index.css     # 전체 스타일
└── js/
    └── index.js      # 게임 전체 로직 (Three.js ES Module)
```

## 기술 스택

- **Three.js r160** — 3D 렌더링 (CDN importmap)
- **Web Audio API** — 효과음 (점프·젬 수집·사망)
- **순수 JS / ES Modules** — 빌드 도구 없음
